  KARMA PLAYER - macOS Installation Instructions
═══════════════════════════════════════════════════════════════

⚠️  STEP 1: DOWNLOAD THE CORRECT VERSION FOR YOUR MAC

Check your Mac's processor:
1. Click Apple menu () → About This Mac
2. Look at the "Chip" or "Processor" line:

   • INTEL processor (2020 and earlier)
     → Download: KarmaPlayer-macOS-Intel.zip

   • APPLE M1/M2/M3 (2020 and newer)
     → Download: KarmaPlayer-macOS-AppleSilicon.zip

⚠️  IMPORTANT: Using the wrong version will show a "Bad CPU type" error!

═══════════════════════════════════════════════════════════════
  STEP 2: FIRST-TIME INSTALLATION
═══════════════════════════════════════════════════════════════

macOS may block this app because it's not signed with an Apple
Developer ID certificate ($99/year). This is NORMAL for open-source
apps distributed outside the Mac App Store.

═══════════════════════════════════════════════════════════════
  HOW TO INSTALL (Choose ONE method)
═══════════════════════════════════════════════════════════════

## METHOD 1: Right-Click (Easiest) ⭐ RECOMMENDED
──────────────────────────────────────────────

1. **Right-click** (or **Control+click**) on `KarmaPlayer.app`
2. Select **Open**
3. A warning will appear saying the app is from an unidentified developer
4. Click **Open**
5. From now on, macOS will remember your choice ✅

---

## METHOD 2: Allow Through macOS Security Settings
(Needed when the dialog has no “Open” button)
──────────────────────────────────────────────

1. Attempt to open `KarmaPlayer.app` normally (double-click)
   → You’ll see “KarmaPlayer.app was blocked…”
2. Open **System Settings**
3. Go to **Privacy & Security**
4. Scroll down until you see:
   **“KarmaPlayer.app was blocked from opening…”**
5. Click **Open Anyway**
6. Confirm by clicking **Open**

After this one-time approval, the app will launch normally.

---

## METHOD 3: Terminal (Rare Edge Case)
──────────────────────────────────────────────

If macOS still refuses:

1. Open **Terminal** (Applications → Utilities → Terminal)
2. Run this command (adjust path if needed):

```

xattr -cr ~/Downloads/KarmaPlayer.app

```

3. Press **Enter**
4. Double-click `KarmaPlayer.app` again

This removes the quarantine flag safely.

---

## NOTE (macOS Gatekeeper Logic)
macOS requires **one** explicit user action to trust apps from outside the
App Store. After that, it stops blocking this specific app bundle.

═══════════════════════════════════════════════════════════════
WHY DOES THIS HAPPEN?
═══════════════════════════════════════════════════════════════

macOS Gatekeeper blocks apps from "unidentified developers" to
protect you from malware. This app is:

✅ Open source (review the code on GitHub)
✅ Built by the community
✅ Safe to use

Apple requires a $99/year Developer membership to notarize apps
and remove this warning. As an open-source project, we don’t pay for that.
